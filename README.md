# intellij-crystal-lang

[![](https://github.com/asedunov/intellij-crystal-lang/actions/workflows/gradle.yml/badge.svg)](https://github.com/asedunov/intellij-crystal-lang/actions/workflows/gradle.yml?query=branch:master)

This was an IntelliJ plugin for Crystal language (https://crystal-lang.org/).

This fork exists to demonstrate modernization and is not the stable version, or ammenable to long-term maintenance. Only advanced programmers should try this version, and I will release any discoveries into their own clean forks as they happen.
